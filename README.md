# Shopping List App

This Flutter application helps users manage their shopping lists efficiently. It includes features such as:

- Adding, editing, and deleting shopping list items.
- Saving the shopping list locally.
- Notifications for pending shopping tasks.
- Displaying the nearest grocery store using GPS.

## Getting Started

To run the app, ensure you have Flutter installed and execute:

```bash
flutter run
```

For more information, visit the [Flutter documentation](https://docs.flutter.dev/).
